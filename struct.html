Struct

Uma struct é constituída de um conjunto de variáveis de tipos variados agrupados, construindo uma única variável com múltiplas informações.

estrutura:
    Endereço : Bytes
    0xff932380 : 61 00 00 00 0A 00 00 00 62 00 00 00
    //          |  char a   |   int b   |  char c   |
    //          |     a     |   10 hex  |     x     |

Percebe-se que há um espaço entre as variáveis do tipo int e char, estes espaços podem estar preenchidos com algum tipo de lixo de memória, isso acontece para que exista um alinhamento de informações na memória.

Este alinhamento é feito de forma a facilitar o acesso aos dados armazenados na struct, pois seria necessário calcular restos e levar em conta o tipo de cada variável para poder acessá-las sequencialmente, no entanto com os endereços sendo múltiplos simples, torna-se mais fácil e eficiente o acesso à informação armazenada.
